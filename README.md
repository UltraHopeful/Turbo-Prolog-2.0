# Turbo-Prolog-2.0
In this you will find some good basic programs of Turbo Prolog 2.0
You will find programs with output of given topics :
Cut predicate
Fail predicate
input -> readln , readchar , readreal , readint predicates
output -> write , writef 
go predicate
